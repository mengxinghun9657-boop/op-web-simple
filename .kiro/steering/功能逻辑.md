## 🔐 一、用户认证与管理模块

### 1.1 功能概述

用户认证采用 JWT Token 机制，支持：
- 用户名/密码登录
- Token 刷新
- 基于角色的权限控制（超级管理员/管理员/普通用户）
- 审计日志记录

### 1.2 后端实现

**文件**: `backend/app/api/v1/users.py`

#### API端点

| HTTP方法 | 路径 | 函数名 | 权限 | 功能描述 |
|---------|------|-------|------|---------|
| POST | `/auth/login` | `login` | 公开 | 用户登录，返回JWT Token |
| POST | `/auth/refresh` | `refresh_token` | 需Token | 刷新Access Token |
| GET | `/users/me` | `get_current_user_info` | 需登录 | 获取当前用户信息 |
| GET | `/users` | `get_users` | admin+ | 获取用户列表 |
| POST | `/users` | `create_user` | admin+ | 创建用户 |
| PUT | `/users/{user_id}` | `update_user` | admin+ | 更新用户信息 |
| DELETE | `/users/{user_id}` | `delete_user` | super_admin | 删除用户 |
| POST | `/users/{user_id}/reset-password` | `reset_password` | admin+ | 重置密码 |
| GET | `/audit-logs` | `get_audit_logs` | admin+ | 获取审计日志 |

#### 核心函数

```python
# 登录验证
def authenticate_user(db: Session, username: str, password: str) -> Optional[User]

# Token生成
def create_access_token(data: dict, expires_delta: Optional[timedelta] = None) -> str

# 权限检查装饰器
def require_admin(current_user = Depends(get_current_user)) -> User
```

## 🔌 九、API调用链汇总

### 9.1 告警管理调用链

```
【告警接收】
HAS系统 → POST /api/v1/has/webhook → receive_has_alert()
  → AlertProcessor.process_alert()
    → AlertParser.parse()
    → ManualMatcher.match()
    → DiagnosisAPI.submit_diagnosis_task() (CCE集群)
    → AIInterpreter.interpret()
    → WebhookNotifier.send_feishu()/send_ruliu()

【告警查询】
前端 → GET /api/v1/alerts → get_alerts()
  → 数据库查询(alert_records) + 关联(diagnosis_results)

【告警诊断】
前端 → POST /api/v1/alerts/{id}/diagnose → trigger_diagnosis()
  → DiagnosisAPI.submit_diagnosis_task()
  → 后台轮询 → DiagnosisAPI.query_diagnosis_result()
  → AIInterpreter.interpret()
  → 更新数据库
```

### 9.2 CMDB调用链

```
【数据同步】
前端 → POST /api/v1/cmdb/sync → sync_from_api()
  → CMDBSyncService.sync_from_amis_api()
    → 调用AMIS API
    → 解析响应
    → 批量插入/更新(iaas_servers, iaas_instances)

【BCE同步】
前端 → POST /api/v1/cmdb/bce/sync → bce_sync()
  → BCESyncService.sync_bcc()
    → _BCCDownloader.download()
    → _LocalDBWriter.import_csv()
  → BCESyncService.sync_cce()
    → _CCEDownloader.download(cluster_id)
    → _LocalDBWriter.import_csv()
```

### 9.3 PFS监控调用链

```
【指标查询】
前端 → POST /api/v1/pfs/query → query_metrics()
  → PFSPrometheusClient.query_metric()
    → 检查Redis缓存
    → 调用Prometheus API
    → 写入Redis缓存(5分钟)
    → 返回数据

【数据导出】
前端 → POST /api/v1/pfs/export → export_data()
  → 创建异步任务(Task)
  → 后台执行查询
  → 生成CSV文件
  → 上传MinIO
  → 更新任务状态
  → 前端轮询 → GET /api/v1/pfs/task/{task_id}
  → 下载 → GET /api/v1/pfs/download/{filename}
```
